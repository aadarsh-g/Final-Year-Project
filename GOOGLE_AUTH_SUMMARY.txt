┌────────────────────────────────────────────────────────────────────────────┐
│ 🔄 AUTHENTICATION FLOW                                                     │
└────────────────────────────────────────────────────────────────────────────┘

1. User clicks "Continue with Google"
   ↓
2. Redirect to Google's login page
   ↓
3. User enters Google credentials
   ↓
4. Google redirects to: /auth/google/callback?code=...
   ↓
5. Frontend exchanges code for user info
   ↓
6. Frontend sends user data to backend
   ↓
7. Backend creates/updates user, returns JWT
   ↓
8. Frontend stores token in localStorage
   ↓
9. Redirect to catalog (user) or admin dashboard (admin)
   ↓
10. ✅ User is logged in!

Step 1: Get Google OAuth Credentials
  → Visit: https://console.cloud.google.com/
  → Create project → Enable OAuth
  → Copy Client ID and Client Secret

Step 2: Create frontend/.env file
  → Add your Google credentials:
     VITE_GOOGLE_CLIENT_ID=your_client_id_here
     VITE_GOOGLE_CLIENT_SECRET=your_client_secret_here
     VITE_API_URL=http://localhost:5001

Step 3: Restart frontend server
  → npm run dev

Step 4: Test!
  → Click "Continue with Google" on login/signup page

NEW ENDPOINTS:
  POST /api/auth/google
    Body: { googleId, email, fullName, avatar }
    Returns: { success, token, user }

  POST /api/auth/google/verify
    Body: { token }
    Returns: { success, message }

  POST /api/auth/login
