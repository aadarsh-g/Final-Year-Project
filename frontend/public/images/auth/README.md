# Authentication Page Illustrations

This folder contains illustrations for the Login and Signup pages.

## Required Images

Place your illustration images here:

### Login Page Illustration
- **Filename**: `login-illustration.png` (or `.jpg`, `.svg`, `.webp`)
- **Recommended size**: 600x500px or larger
- **Description**: Secure login illustration showing a person with browser/form interface

### Signup Page Illustration
- **Filename**: `signup-illustration.png` (or `.jpg`, `.svg`, `.webp`)
- **Recommended size**: 600x500px or larger
- **Description**: Secure signup illustration showing a person with browser/form interface

## Image Guidelines

- **Format**: PNG (recommended for transparency), JPG, SVG, or WebP
- **Size**: Keep under 500KB for optimal performance
- **Style**: Should match the design aesthetic (teal/turquoise accents, clean lines)
- **Content**: Person interacting with secure form/browser window with lock icon

## How It Works

The pages will:
1. Try to load the image from this folder
2. If the image doesn't exist, display a fallback SVG illustration
3. Maintain the teal circular background behind the illustration
